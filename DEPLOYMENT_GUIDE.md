# How to Enable GitHub Pages

To make your website live, follow these steps:

1.  **Go to your Repository Settings**:
    - Open [https://github.com/aleenamalik12354-bot/fg-pg-website-interface/settings](https://github.com/aleenamalik12354-bot/fg-pg-website-interface/settings)

2.  **Navigate to Pages**:
    - On the left sidebar, click on **Pages** (under the "Code and automation" section).

3.  **Configure Source**:
    - Under **Build and deployment** > **Source**, select **Deploy from a branch**.
    - Under **Branch**, select **main** (or `gh-pages` if you prefer).
    - Ensure the folder is set to **/(root)**.
    - Click **Save**.

4.  **Wait for Deployment**:
    - GitHub will take a minute or two to build your site.
    - Refresh the page. You should see a message saying "Your site is live at...".

5.  **View Your Site**:
    - Your website will be available at: [https://aleenamalik12354-bot.github.io/fg-pg-website-interface/](https://aleenamalik12354-bot.github.io/fg-pg-website-interface/)
