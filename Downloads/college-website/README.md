## ğŸ”§ Customization Guide

### Changing Colors
Edit `assets/style.css`:
```css
/* Change primary color */
header { background: #YOUR_COLOR; }
.home-text h1 { color: #YOUR_COLOR; }
```

### Updating Slider Images
1. Add your images to `assets/images/`
2. Update `index.html`:
```html
<div class="slides fade">
    <img src="assets/images/your-image.jpg">
</div>
```

### Adjusting Slider Speed
Edit `assets/script.js`:
```javascript
// Change 3000 to your desired milliseconds
setTimeout(showSlides, 3000);
```

### Adding New Pages
1. Create new HTML file (e.g., `gallery.html`)
2. Add navigation link in all pages:
```html
<a href="gallery.html">Gallery</a>
```
